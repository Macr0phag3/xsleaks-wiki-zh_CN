## 部署流程

### 在本地部署

1. 安装 [Hugo 框架](https://gohugo.io/getting-started/installing/) **补充** 版本需要 > 0.68
2. 克隆本仓库
3. 在根目录中运行 `hugo server --minify`
4. 访问 http://localhost:1313 (或者按照 hugo 的输出来访问)

### 生成静态文件

1. 执行 `hugo --buildDrafts`
